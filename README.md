# Workspace Platform

A SharePoint-inspired workspace platform built as a learning-focused software engineering project.

The goal is to design and develop a modern platform where teams can organize files, create pages, build dashboards using widgets, search content, collaborate in real time, and interact with AI-powered features.

## Features

* Workspaces
* Files and Folders
* Widget-Based Pages
* Search
* Real-Time Updates
* AI Assistant
* Role-Based Access Control

## Goals

* Build a realistic multi-user application
* Learn modern frontend and backend architecture
* Practice software design, testing, and project planning
* Explore AI integration without relying on external frameworks
